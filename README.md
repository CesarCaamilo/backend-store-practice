## Setup

1. If you donâ€™t have Node.js installed, [install it from here](https://nodejs.org/en/) (Node.js version >= 16.0.0 required)

2. Clone this repository

3. Navigate into the project directory

   ```bash
   $ cd backend-store-practice
   ```

4. Install the requirements

   ```bash
   $ npm install
   ```

5. Make a copy of the example environment variables file

   On Linux systems:
   ```bash
   $ cp .env.example .env
   ```
   On Windows:
   ```powershell
   $ copy .env.example .env
   ```

6. Run the app

   ```bash
   $ npm run dev
   ```

You should now be able to access the app at [http://localhost:2002](http://localhost:2002).
